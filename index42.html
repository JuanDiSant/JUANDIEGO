HTML: El lenguaje de marcado estándar para crear páginas web. Permite estructurar el contenido de una página mediante etiquetas.

Etiqueta: En HTML, una etiqueta es un elemento utilizado para marcar el inicio y el fin de un elemento en la página web.

Elemento: En HTML, un elemento es una construcción que consta de una etiqueta de inicio, contenido y una etiqueta de cierre.

Atributo: Un atributo proporciona información adicional sobre un elemento HTML y generalmente se especifica en la etiqueta de inicio.

Enlace: Un enlace o hipervínculo es un elemento que permite a los usuarios navegar de una página a otra con solo hacer clic en él.

Imagen: Un elemento img se utiliza para insertar imágenes en una página web.

Lista: Una lista es un conjunto de elementos relacionados que se presentan en orden secuencial, como una lista ordenada o desordenada.

Formulario: Un formulario es un conjunto de campos de entrada que permite a los usuarios enviar datos a un servidor web para su procesamiento.

Tabla: Una tabla es una estructura de datos bidimensional que organiza la información en filas y columnas.

CSS: Cascading Style Sheets (CSS) es un lenguaje de diseño utilizado para controlar el aspecto y el formato del contenido HTML en una página web.
